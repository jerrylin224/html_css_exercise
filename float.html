Below see a list of issues and side effects with BFC-creating properties on the main area. Best solution: overflow: hidden This is the only reliably working property, use it whenever you can. Caution: This will literally cut off any content that leaves the container. You must not specify a height on the main area so it can stretch as needed. If any content ever needs to overlap the main area's borders, try adding padding (no pun intended).
